Formularios

En esta nueva agina se van a ver algunas etiquetas que permiten a la página interactuar con el usuario, para ello se va a hacer uso de los "formularios" y las etiquetas que se pueden colocar dentro de estos

Para iniciar, se debe abrir la etiqueta de los formularios: form, dentro de esta etiqueta se colocan otras dos etiquetas:

label:
Nombre de la variable
input
Entrada de esa variable

Si se va a crear más de un elemento en el formulario, se debe separar con br y en secciones, hay diversos tipos de input:

Input

  • input type="text"
  • input type="password"
  • input type="tel"
  • input type="email"
  • input type="url"
  • input type="number"
  • input type="date"
  • input type="color"
  • input type="search"
  • input type="range"

Cuadro de texto

En los formularios también existe una etiqueta la cual es textarea y sirve para dejar un cuadro en donde el usuario puede escribir

Este cuadro puede ser ajustado al tamaño deseado usando las etiquetas rows(filas) o cols (columnas)


Botones en formularios

Para poder recibir la información que el usuario está llenando se utilizan los botones de enviar, y para indicar a donde se quieren enviar esos datos se usa la etiqueta action y la etiqueta method en la apertura del formulario form action="" method="" donde action contendrá la ruta del servidor y method el "método" a usar: get o post

Hay dos formas de crear un botón, con el input type="submit" o con la etiqueta button, esta última tiene más usos y es más personalizable, por ejemplo, se puede clocar una imagen de fondo
